# 🤖 AI Agent - GitHub/GitLab Entegrasyonu

## 📋 Nasıl Çalışır?

### Akış

```
1. GitHub/GitLab repo → TaskNebula project'e bağlanır
2. GitHub issue'lar → TaskNebula'ya sync olur
3. TaskNebula'da "Start AI Agent" → Agent başlar
4. Agent Docker container'da → GitHub repo'yu clone eder
5. Agent kod yazar → Commit + Push yapar
6. Agent PR oluşturur → GitHub'a
7. PR merge olunca → Webhook ile issue kapanır
```

### 3. **Agent Execution'da Repo Clone**

**Worker Update:** `/workers/agent-worker.ts`

```typescript
async function handleAgentSetup(job, data) {
  // 1. Fetch repository from project
  const repo = await db.query.githubRepositories.findFirst({
    where: eq(githubRepositories.projectId, data.projectId)
  });

  // 2. Fetch GitHub token
  const installation = await db.query.githubInstallations.findFirst({
    where: eq(githubInstallations.id, repo.installationId)
  });

  // 3. Create workspace directory
  const workspacePath = `/tmp/workspaces/${data.workspaceId}`;
  await fs.mkdir(workspacePath, { recursive: true });

  // 4. Clone repository
  await gitOperations.cloneRepository(
    repo.cloneUrl,
    workspacePath,
    installation.accessToken
  );

  // 5. Update workspace in DB
  await db.update(agentWorkspaces).set({
    status: 'ready',
    workingDirectory: workspacePath,
    setupCompletedAt: new Date()
  });
}
```

### 4. **PR Auto-Create**

**Octokit Integration:**

```typescript
// /lib/github/pr-manager.ts
import { Octokit } from '@octokit/rest';

export async function createPullRequest({
  owner,
  repo,
  branchName,
  title,
  body,
  accessToken
}) {
  const octokit = new Octokit({ auth: accessToken });

  const pr = await octokit.pulls.create({
    owner,
    repo,
    title,
    head: branchName,
    base: 'main',
    body: `${body}\n\n🤖 Auto-generated by TaskNebula AI Agent`,
  });

  return pr.data;
}
```

**Executor Update:**

```typescript
// After push, create PR
if (commitSha) {
  const pr = await createPullRequest({
    owner: repo.owner,
    repo: repo.name,
    branchName,
    title: context.issueTitle,
    body: context.issueDescription,
    accessToken: installation.accessToken
  });

  // Save PR to database
  await db.insert(githubPullRequests).values({
    repositoryId: repo.id,
    issueId: context.issueId,
    prNumber: pr.number,
    prUrl: pr.html_url,
    state: 'open'
  });
}
```

## 📝 Setup Adımları (Manuel)

### Adım 1: GitHub Personal Access Token Oluştur

1. GitHub → Settings → Developer settings → Personal access tokens
2. "Generate new token (classic)" tıkla
3. Scopes:
   - `repo` (full control)
   - `workflow`
   - `read:org`
4. Token'ı kopyala: `ghp_xxxxxxxxxxxxx`
